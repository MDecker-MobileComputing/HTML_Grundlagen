Aufklappbare Details

Diese Seite überprüft mit Modernizr (JavaScript-Bibliothek unter MIT-Lizenz) zur Laufzeit, ob der Browser den details-Tag unterstützt. Wenn der Tag nicht unterstützt wird, dann wird mit einem Dialog eine Warnung ausgegeben. Die Microsoft-Browser IE und Edge unterstützen diesen Tag z.B. nicht, siehe auch diesen Eintrag auf caniuse.com.

Abkürzungen zur Android-Programmierung:

ADB

Android Debug Bridge

APK

Android Package; Installations-Datei einer Android-App.

DVM

Dalvik Virtual Machine; führt den DEX-Bytecode einer Android-App aus.
